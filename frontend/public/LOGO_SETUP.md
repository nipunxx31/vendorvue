# Custom Logo for vendorvue

Place your logo file here: `logo.png`

## Logo Specifications:
- **Format:** PNG with transparent background recommended
- **Size:** 256x256 pixels (minimum)
- **Aspect Ratio:** Square (1:1) works best
- **File size:** Keep under 100KB

## How to add your logo:
1. Create or download your logo as a PNG file
2. Rename it to `logo.png`
3. Place it in this directory (`frontend/public/`)
4. The logo will automatically appear on:
   - Landing Page
   - Admin Login
   - Customer Login
   - Vendor Login
   - All Dashboards

The logo uses a fallback to text if the image file is missing, so the site will work fine even without the logo image initially.
